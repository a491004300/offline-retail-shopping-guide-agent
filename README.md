# 线下零售导购智能体
本项目旨在打造一款面向线下高标准化产品零售场景、服务于坚持透明化经营的优质实体商家（以3C数码等品类代理商门店为核心）的中立导购智能体，核心覆盖汽车、3C数码、大家电、标准化家居等品类。 智能体以打破线下消费信息壁垒、解决行业根源性信任危机为核心使命，通过AR智能导购眼镜、智能引导平板等多终端适配的交互形态，基于门店真实的产品体系、价格体系、活动政策与实时库存数据，为到店消费者提供场景化、沉浸式的透明购物体验，实现需求匹配、全透明商品信息与优惠方案演算，将线上透明化购物体验完整平移至线下。    由智能体全权承接信任建立、需求-预算最优匹配、信息透明化交付，彻底解决消费者决策焦虑，大幅节省人工成本、提升人效；  由门店人员聚焦产品体验、售后保障、技术维护等增值服务环节，放大服务价值。
